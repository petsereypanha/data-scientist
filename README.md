## Getting Started

### Prerequisites
- Python 3.x
- NumPy
- Pandas
- Matplotlib

### Installation
```bash
pip install numpy pandas matplotlib
```

## Learning Path

1. Start with **introduction-python/** to build Python fundamentals
2. Progress to **intermediate-python/** for data analysis tools
3. Master **manipulation-pandas/** for advanced data manipulation

## Topics Covered

- Python programming basics
- Data structures (lists, dictionaries, arrays)
- NumPy for numerical computing
- Pandas for data manipulation
- Data visualization with Matplotlib
- Statistical analysis and simulations
- DataFrame operations and transformations
